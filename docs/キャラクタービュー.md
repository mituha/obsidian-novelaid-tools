# キャラクタービュー

## 概要

Obsidianのサイドバーに、小説に登場するキャラクターの一覧を表示する専用のビューを追加します。
このビューから、キャラクターの設定ファイルに素早くアクセスできるようになり、執筆効率の向上を目的とします。
キャラクターは、ファイルのメタデータ（フロントマター）に特定のタグが含まれていることで識別されます。

## 仕様

### 1. 基本機能

- **ビューの追加**: Obsidianの左側サイドバーに、新しいアイコン（例：人物アイコン）を追加し、クリックするとキャラクタービューが開くようにします。
- **キャラクターの識別**:
    - 設定で指定されたタグ（デフォルト: `Character`, `登場人物`, `人物`）がメタデータに含まれるファイルをキャラクターとして認識します。
    - `tags: [登場人物]` や `tags: 登場人物` の両方の形式に対応します。
- **一覧表示**:
    - 認識されたキャラクターのファイル名を一覧で表示します。
    - ファイル名をクリックすると、そのキャラクターのファイルをアクティブなエディタで開きます。

### 2. 表示内容

- **基本表示**:
    - 各キャラクターのファイル名（拡張子`.md`は除く）を表示します。
- **追加情報の表示（オプション）**:
    - ファイルのメタデータからエイリアス（`aliases`）や説明（`description`）を読み込み、ファイル名の下に小さく表示する機能を設けます。
    - 表示するメタデータのキーは、設定画面でカスタマイズ可能です。
- **アイコン/画像の表示（将来的な展望）**:
    - メタデータに画像ファイルのパスが指定されている場合、その画像（キャラクターの肖像など）をファイル名の横にサムネイル表示します。

### 3. ビューの操作

- **並び替え**:
    - ファイル名順（昇順/降順）
    - ファイル更新順（新しい順/古い順）
- **フィルタリング**:
    - ビューの上部に検索ボックスを設置し、入力されたテキストでキャラクター名を絞り込み表示します。
- **右クリックメニュー**:
    - `ファイルを開く`
    - `新しいタブで開く`
    - `ファイル名を変更`
    - `ファイルを削除`
    - `ファイルが格納されているフォルダを開く`

### 4. 設定項目

プラグインの設定画面に、以下の項目を追加します。

- **認識用タグ**: キャラクターとして認識するためのタグをユーザーが自由に追加・編集・削除できるようにします。（例: `char`, `主要人物` など）
- **追加情報のメタデータキー**: 一覧に表示する追加情報（説明など）を取得するためのメタデータキーをユーザーが指定できるようにします。（デフォルト: `description`）
