# 地理ビュー

## 概要

Obsidianのサイドバーに、小説に登場する地名の一覧を表示する専用のビューを追加します。
このビューから、地名の設定ファイルに素早くアクセスできるようになり、執筆効率の向上を目的とします。
地名は、ファイルのメタデータ（フロントマター）に特定のタグが含まれていることで識別されます。

## 仕様

### 1. 基本機能

- **ビューの追加**: Obsidianの左側サイドバーに、新しいアイコン（例：地図アイコン）を追加し、クリックすると地理ビューが開くようにします。
- **地名の識別**:
    - 設定で指定されたタグ（デフォルト: `Geography`, `地理`, `地名`, `Geo`）がメタデータに含まれるファイルを地名として認識します。
    - `tags: [地理]` や `tags: 地理` の両方の形式に対応します。
- **一覧表示**:
    - 認識された地名のファイル名を一覧で表示します。
    - ファイル名をクリックすると、その地名のファイルをアクティブなエディタで開きます。

### 2. 表示内容

- **基本表示**:
    - 各地名のファイル名（拡張子`.md`は除く）を表示します。
- **追加情報の表示（オプション）**:
    - ファイルのメタデータからエイリアス（`aliases`）や説明（`description`）を読み込み、ファイル名の下に小さく表示する機能を設けます。
- **アイコン/画像の表示（将来的な展望）**:
    - メタデータに画像ファイルのパスが指定されている場合、その画像（地図や風景など）をファイル名の横にサムネイル表示します。

### 3. ビューの操作

- **並び替え**:
    - ファイル名順（昇順/降順）
    - ファイル更新順（新しい順/古い順）
- **フィルタリング**:
    - ビューの上部に検索ボックスを設置し、入力されたテキストで地名で絞り込み表示します。
- **右クリックメニュー**:
    - `ファイルを開く`
    - `新しいタブで開く`
    - `ファイル名を変更`
    - `ファイルを削除`
    - `ファイルが格納されているフォルダを開く`
